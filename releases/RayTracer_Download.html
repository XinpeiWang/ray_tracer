🚀 Quick Start

  1. Extract the ZIP file to any folder
  2. Open the RayTracer_Package folder
  3. Double-click RayTracerGUI.exe
  4. Select "Preview" quality and "800x800" resolution
  5. Click "Start Render" and wait ~30 seconds
  6. Your image opens automatically!
